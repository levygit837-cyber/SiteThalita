# Procedimentos — Dra. Thalita Barbosa

Arquivo-base para organização das categorias e subcategorias de procedimentos no site.

> **Observação estratégica:** os valores foram mantidos como referência interna, pois vieram da tabela original. Para o site público, recomenda-se avaliar se os preços devem aparecer ou se devem ser substituídos por CTA de avaliação/agendamento.

---

## 1. Estrutura recomendada para o site

### Camada 1 — Cards de categorias
Cada card representa uma **categoria principal** de tratamento.

Sugestão de informações no card:

- Nome da categoria
- Frase curta de benefício
- Ícone discreto
- Número de opções/protocolos disponíveis
- CTA: `Ver opções`

Exemplo:

```text
Toxina Botulínica
Suavização de rugas, equilíbrio facial e tratamentos terapêuticos.
8 opções disponíveis
[Ver opções]
```

### Camada 2 — Subcategorias dentro do card expandido
Ao clicar no card da categoria, o usuário visualiza uma lista limpa com os procedimentos daquela categoria.

Sugestão de informações para cada subcategoria:

- Nome do procedimento
- Área ou indicação principal
- Breve descrição em 1 ou 2 linhas
- CTA secundário: `Saiba mais`

### Camada 3 — Página ou seção detalhada do procedimento
Para procedimentos importantes, recomenda-se criar uma página própria ou uma seção detalhada com link direto.

Estrutura sugerida para cada procedimento:

1. O que é
2. Para quem é indicado
3. Como funciona
4. Áreas tratadas
5. Duração média e recuperação
6. Cuidados antes e depois
7. Perguntas frequentes
8. CTA para avaliação

# 4. Recomendação estratégica final

A melhor experiência para o paciente é usar um modelo híbrido:

1. **Cards de categorias** para visão geral.
2. **Expansão do card** para mostrar subcategorias sem tirar o usuário da página.
3. **Página de detalhe** para procedimentos principais, com conteúdo educativo, SEO e CTA.
4. **CTA fixo de agendamento** no desktop e no mobile.
5. **Filtros por objetivo do paciente**, não apenas por nome técnico.

Exemplos de filtros por objetivo:

- Quero suavizar rugas
- Quero melhorar a qualidade da pele
- Quero tratar suor excessivo
- Quero melhorar o contorno facial
- Quero cuidar do cabelo
- Quero um protocolo personalizado
