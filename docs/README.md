## Visão Geral do Projeto

- **Segmento:** Medicina Estética / Clínica de Estética
- **Público-alvo:** Pacientes em busca de tratamentos estéticos minimamente invasivos
- **Tom de voz:** Sofisticado, acolhedor, científico, natural
- **Tema visual:** Layout claro (branco/rosé) com acentos em rosé médio
